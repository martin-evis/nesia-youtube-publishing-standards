# NESIA TALKSHOW

Tento priečinok definuje štandard pre publikovanie NESIA TALKSHOW videí.

## Definícia formátu

TALKSHOW je rozhovorový formát, v ktorom NESIA komunikuje hodnoty, know-how, inšpiráciu a témy súvisiace so značkou, bývaním, interiérom, zákazníckou skúsenosťou, dizajnom, predajom, službami alebo zákulisím firmy.

## Výstup pre každú epizódu

Každá epizóda má mať pripravený publikačný balík:

- nadpis,
- alternatívne nadpisy,
- popis,
- hashtagy,
- tagy,
- kapitoly,
- thumbnail text,
- pripnutý komentár,
- odporúčané nastavenia v YouTube Studio,
- kontrolné poznámky.
