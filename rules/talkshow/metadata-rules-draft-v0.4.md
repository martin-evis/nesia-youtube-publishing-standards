# Metadata rules – NESIA TALKSHOW draft v0.4

Status: pracovný draft.

## Hashtagy

Odporúčanie pre TALKSHOW:

- 3 hlavné hashtagy,
- maximálne 5 hashtagov v bežnom popise,
- nepoužívať nesúvisiace trendy,
- nepoužívať všeobecné hashtagy bez hodnoty.

Pracovný základ:

```text
#NESIA #NESIATalkshow #AndreaVadkerti
```

Potom doplniť 1–2 tematické hashtagy podľa epizódy, napríklad:

```text
#UmelaInteligencia
#Bata
#ZakaznickaPsychologia
#Materstvo
```

## Tagy / štítky

Tagy sú doplnkové. Majú pomôcť pri alternatívnych zápisoch, menách a témach, nie suplovať kvalitný názov a popis.

Odporúčaný rozsah:

- meno hosťa,
- celé meno moderátorky,
- hlavná téma,
- alternatívne zápisy mena alebo témy,
- značka NESIA,
- názov formátu.

## Kapitoly

Ak video má viac ako 25–30 minút, kapitoly sú odporúčané. Minimálne:

```text
00:00 Úvod
...
```

Kapitoly majú pomenovať reálne témy, nie všeobecné frázy typu „Rozhovor pokračuje“.
