# Description rules – NESIA TALKSHOW draft v0.4

Status: pracovný draft.

## Cieľ popisu

Popis má divákovi rýchlo vysvetliť hodnotu epizódy, zachytiť dôležité témy a pomôcť YouTube aj ľuďom pochopiť kontext videa.

## Odporúčaná štruktúra

```text
1. Silný úvodný odsek – o čom je rozhovor a prečo stojí za pozornosť.
2. Predstavenie hosťa a moderátorky.
3. Hlavné témy epizódy v bodoch.
4. CTA: odber, komentár, sledovanie ďalších epizód.
5. Odkazy: web, sociálne siete, playlist.
6. Hashtagy.
```

## Štýl

- písať prirodzene,
- nezačínať každú epizódu rovnakou frázou,
- nekopírovať iba názov do prvého odseku,
- nepoužívať prázdne superlatívy,
- pri hodnotových témach používať pokojný a dôstojný jazyk.

## Pozorovanie z Baťa epizódy

Popis pri epizóde s Gabrielou Končitíkovou funguje, pretože vytvára hodnotový oblúk:

```text
výkon a hodnoty, zisk a zmysel, disciplína a ľudskosť
```

Tento typ kontrastov je pre NESIA TALKSHOW veľmi vhodný, ak vychádza z reálneho obsahu videa.

## Čo doplniť po získaní transcriptov

- typické slová moderátorky,
- typické CTA,
- opakujúce sa hodnoty NESIA,
- optimálnu dĺžku popisu podľa skutočných publikovaných príkladov.
