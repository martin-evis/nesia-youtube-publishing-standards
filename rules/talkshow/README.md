# TALKSHOW rules

Tento priečinok obsahuje pracovné pravidlá pre NESIA TALKSHOW.

## Stav

Súbory označené ako `draft-v0.4` nie sú finálne pravidlá. Sú odvodené zo 6 publikovaných videí, ale zatiaľ chýbajú transcripty a viaceré pôvodné YouTube popisy.

Finálne pravidlá budú uložené bez slova `draft`, napríklad:

```text
voice-of-tone.md
title-rules.md
description-rules.md
metadata-rules.md
thumbnail-rules.md
```
