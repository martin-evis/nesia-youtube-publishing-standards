# Repository workflow

Tento dokument popisuje, ako má tím pracovať s repozitárom pri tvorbe publikačných podkladov pre YouTube videá.

## 1. Pridanie nového videa

Pre každé nové video vytvor samostatný metadata package:

```text
metadata-packages/talkshow/XXX-tema-hosta.md
```

Odporúčané pomenovanie:

```text
001-meno-hosta-hlavna-tema.md
002-meno-hosta-hlavna-tema.md
```

Ak nie je známe poradové číslo, použi pracovný dátum:

```text
2026-06-14-talkshow-pracovny-nazov.md
```

## 2. Vstupy pred tvorbou metadát

Minimálne vstupy:

- pracovná URL videa alebo názov súboru,
- názov série: NESIA TALKSHOW,
- meno hosťa,
- pozícia / firma hosťa,
- hlavná téma epizódy,
- 3 až 7 kľúčových bodov epizódy,
- prepis videa alebo poznámky z rozhovoru,
- hlavné CTA,
- odkazy, ktoré majú byť v popise,
- playlist, do ktorého má video patriť.

## 3. Tvorba publikačného balíka

Pre každé video priprav:

- primárny nadpis,
- 5 alternatívnych nadpisov,
- krátky popis do prvých 2 riadkov,
- dlhší popis,
- kapitoly / timestampy,
- hashtagy,
- tagy / štítky,
- kategóriu,
- návrh thumbnail textu,
- návrh pripnutého komentára,
- návrh end screenu a cards.

## 4. Review

Pred publikovaním musí prebehnúť kontrola:

- meno hosťa je správne,
- firma / pozícia je správna,
- nadpis nie je zavádzajúci,
- popis neobsahuje neoverené tvrdenia,
- odkazy fungujú,
- hashtagy sú relevantné,
- video je zaradené do správneho playlistu,
- thumbnail zodpovedá téme videa.

## 5. Publikovanie

Po schválení sa metadáta prenesú do YouTube Studio.

Odporúčaný postup:

1. Upload videa.
2. Vložiť nadpis.
3. Vložiť popis.
4. Pridať thumbnail.
5. Zaradiť do playlistu.
6. Nastaviť audience / made for kids.
7. Vyplniť advanced settings.
8. Pridať titulky alebo skontrolovať automatické titulky.
9. Pridať end screen / cards.
10. Nastaviť visibility alebo schedule.

## 6. Po publikovaní

Do metadata package doplň:

- finálnu URL videa,
- dátum zverejnenia,
- finálny nadpis,
- finálny popis,
- použitý thumbnail,
- prípadné rozdiely oproti návrhu,
- poznámky pre ďalšiu epizódu.
