# Pre-publication checklist

Pred zverejnením TALKSHOW videa skontroluj:

## Obsah

- [ ] Meno hosťa je správne.
- [ ] Pozícia / firma hosťa je správna.
- [ ] Hlavná téma zodpovedá obsahu videa.
- [ ] Nadpis nie je zavádzajúci.
- [ ] Popis neobsahuje neoverené tvrdenia.
- [ ] V popise nie sú interné poznámky typu `DOPLNIŤ`.

## YouTube limity

- [ ] Nadpis má max. 100 znakov.
- [ ] Popis má max. 5 000 znakov.
- [ ] Hashtagy nemajú medzery.
- [ ] Hashtagy sú relevantné.
- [ ] Tagy nie sú vložené do popisu ako spam.

## Popis

- [ ] Prvé dve vety dávajú zmysel bez ďalšieho kontextu.
- [ ] Popis obsahuje, čo sa divák dozvie.
- [ ] Kapitoly sú doplnené alebo vedome vypnuté.
- [ ] Playlist je uvedený.
- [ ] Odkazy fungujú.
- [ ] CTA je jasné a nie je ich príliš veľa.

## Thumbnail

- [ ] Thumbnail je nahraný.
- [ ] Text je čitateľný na mobile.
- [ ] Vizuál zodpovedá obsahu videa.
- [ ] Thumbnail je konzistentný so sériou.

## YouTube Studio nastavenia

- [ ] Playlist je nastavený.
- [ ] Category je nastavená.
- [ ] Language je nastavený.
- [ ] Audience / made for kids je skontrolované.
- [ ] Paid promotion je skontrolované.
- [ ] Altered or synthetic content je skontrolované.
- [ ] Comments a ratings sú nastavené podľa stratégie kanála.
- [ ] Embedding / distribution je skontrolované.
- [ ] End screen je nastavený, ak je relevantný.
- [ ] Cards sú nastavené, ak sú relevantné.
- [ ] Subtitles / captions sú skontrolované.

## Finálne schválenie

- [ ] Metadata package je uložený v repozitári.
- [ ] Zodpovedná osoba schválila finálny nadpis.
- [ ] Zodpovedná osoba schválila finálny popis.
- [ ] Video je pripravené na publikovanie alebo naplánovanie.
