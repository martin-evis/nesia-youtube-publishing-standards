# Post-publication checklist

Po zverejnení TALKSHOW videa skontroluj:

## Verejná kontrola

- [ ] Video je dostupné na finálnej URL.
- [ ] Nadpis sa zobrazuje správne.
- [ ] Popis sa zobrazuje správne.
- [ ] Hashtagy sa zobrazujú správne.
- [ ] Thumbnail sa zobrazuje správne.
- [ ] Video je v správnom playliste.
- [ ] Kapitoly fungujú, ak boli pridané.
- [ ] Odkazy v popise fungujú.
- [ ] Pripnutý komentár je pridaný, ak bol plánovaný.

## Interné doplnenie

- [ ] Do metadata package je doplnená finálna URL.
- [ ] Do metadata package je doplnený dátum publikovania.
- [ ] Sú zapísané odchýlky oproti návrhu.
- [ ] Sú zapísané poznámky pre ďalšie video.

## Kontrola po pár dňoch

- [ ] Skontrolovať CTR thumbnailu, ak je dostupné.
- [ ] Skontrolovať retenciu úvodu videa.
- [ ] Skontrolovať komentáre a otázky divákov.
- [ ] Vyhodnotiť, či nadpis / thumbnail netreba upraviť.
- [ ] Zapísať learnings do `video-types/talkshow/examples/published-videos-analysis.md`.
